Alerts

Proporcione mensajes de comentarios contextuales para las acciones típicas del usuario con el puñado de mensajes de alerta disponibles y flexibles.

Ejemplos

Los Alerts están disponibles para cualquier longitud de texto, así como un botón de descarte opcional. Para un estilo adecuado, utilice una de las ocho clases contextuales necesarias (por ejemplo, .alert-succes). Para el despido en línea, utilice el complemento de alertas jQuery..alert-success
