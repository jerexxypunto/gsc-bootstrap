Introducción

GSC-Bootstrap es un framework construido a partir de bootstrap 4.

Alerta: El propósito de construir esta edición de bootstrap es sólo con el propósito de agilizar mi proceso de trabajo como Desarrollador web, no obstante si algún otro Desarrollador web desea hacer uso de esta edición de bootstrap puede hacerlo, pues, es un repositorio publico.

La idea basica es tomar todas las utlidades de bootstrap 4 e implementar las que en mi proceso de trabajo hubiese deseado que tuviese bootstrap.

Breackpoints

Los puntos de quiebre (breackpoint) son:

  • $sm:576px;
  • $md:768px;
  • $lg:992px;

La Grid de GSC-Bootstrap

Bootstrap 4 trabaja una grilla con flexbox.

GSC-Bootstrap implementa una grilla con CSS Grid layout.

CSS Grid en Bootstrap

En CSS grid a diferencia de flexbox las columnas se declaran en el elemento padre, no en los elementos hijos.

Para hacer que un contenedor sea de tipo grid sea display:grid; use la clase .contenedor-grid.

Use la clase .grid-col-n° para definir cuantas columnas tendra su contenendor grid.

Cada clase .grid-col-n° funciona con su propio punto de quiebre...

...Por ejemplo:

La clase .grid-col-2 genera dos columnas, en cualquier tamaño de pantalla.

La clase .grid-col-sm-2 genera dos columnas, pero sólo desde un min-width:576px.

La clase .grid-col-md-2 genera dos columnas, pero sólo desde un min-width:768px.

La clase .grid-col-lg-2 genera dos columnas, pero sólo desde un min-width:992px.

La clase .grid-col-xs-2 genera dos columnas, pero sólo desde un min-width:1200px.

IMPORTANTE

los valores de .grid-col-n° pueden ir de 1 hasta 12.

Por ejemplo:

.grid-col-1 .grid-col-2 .grid-col-3 ... .grid-col-12

Y de igual forma en sus respectivos breackpoints.

La clase .max-content-in-vw se usa en conjuto con la clase .contenedor-grid y dota al contenedor de tantas columnas como quepan en ancho (width) del mismo contenedor.

Filas

Use la clase .grid-row-n°-of-n°px para definir cuantas filas tendra su contenedor grid y cuanto medira cada celda.

Por ejemplo

La clase .grid-row-2-of-150px define que la grid tendra 2 filas y que cada celda de la grid tendra 150px de altura.

A diferencia de la clase .grid-col-n°, la clase .grid-row-n°-of-n°px no es sensible a los breackpoints. Esto quiere decir que presindientdo de tamaño de pantalla las esta propiedad permanecera igual.

IMPORTANTE

los valores de .grid-row-n°-of-n°px pueden ir de 1 hasta 10.
Los valores desde el primer numero van de 1 a 10.

  • .grid-row-2-of-100px
  • .grid-row-3-of-100px
  • .grid-row-10-of-100px

Los valores desde el segudno numero van de 100px a 500px

Por ejemplo:

.grid-col-1 .grid-col-2 .grid-col-3 ... .grid-col-12

Y de igual forma en sus respectivos breackpoints.

Las clases css gap-n° tambien se utilizan solo en compañia de .contenedor-grid y determinan el espaciado entre los grid items:

  • .gap-5
  • .gap-10
  • .gap-15
  • .gap-20
  • .gap-25
  • .gap-30
  • .gap-35
  • .gap-40
  • .gap-45
  • .gap-50
